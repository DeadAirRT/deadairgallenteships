# DeadAir Gallente Ships

## Adds 7 L ships
- Brutix
- Hyperion
- Megathron
- Obelisk
- Proteus
- Talos
- Thorax
## Adds 2 XL ships
- Erebus
- Thanatos
## Required Mods
- DeadAir Faction Galdente
- DeadAir Factions VRO
## Installation
- folder must be named "deadairgallenteships"
- folder must be in game/extensions folder, not documents
